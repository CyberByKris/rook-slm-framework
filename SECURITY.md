# Security Policy

## Public Repository Security Scope

This repository is intentionally sanitized. It should not contain:

- Secrets or credentials
- API keys
- Environment files
- Private prompts
- Customer-specific information
- Internal company materials
- Proprietary implementation details
- Sensitive security procedures

## Reporting a Concern

If you identify sensitive content in this repository, open a private issue or contact the repository owner directly so it can be removed.

## Safe Contribution Guidance

Before contributing, verify that your proposed change does not disclose:

- Private deployment details
- Exploit-enabling information
- Sensitive customer context
- Restricted data flows
- Security control bypass methods
- Unapproved third-party material
