# Diagrams

Place public, sanitized ROOK SLM™ diagrams in this folder.

Recommended diagrams:

- High-level reference architecture
- Boundary-first control model
- Controlled API orchestration flow
- Governance and audit flow
- Persona routing conceptual flow

Avoid uploading diagrams that disclose:

- Customer environments
- Internal network details
- Private API endpoints
- Secrets or credentials
- Proprietary implementation logic
- Sensitive operational workflows
