# Contributing

Contributions should preserve the public, sanitized nature of this repository.

## Contribution Rules

Please do not contribute:

- Proprietary implementation code
- Customer-specific artifacts
- Internal company materials
- Secrets or credentials
- Private system prompts
- Sensitive operational workflows
- Unlicensed third-party content

## Preferred Contributions

Good contributions include:

- Clarifying language
- Public governance templates
- Conceptual architecture improvements
- Sanitized diagrams
- Executive-ready checklists
- Risk framing improvements

## Review Standard

All contributions should support boundary-first, governed AI design without increasing disclosure risk.
