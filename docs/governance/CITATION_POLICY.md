# Citation Policy

Citation policy defines how Rook SLM should attach evidence to claims.

Each cited source should expose the metadata defined in [Provenance Model](PROVENANCE_MODEL.md). Source preference is governed by [Source Hierarchy](SOURCE_HIERARCHY.md).

If a citation cannot be verified, remove it or mark it as unverified. Do not allow fabricated references.

Citation review should confirm:

- The source exists.
- The source metadata is complete enough for review.
- The source supports the claim being made.
- The claim does not overstate the source.
- Date-sensitive claims are checked against the current date.
