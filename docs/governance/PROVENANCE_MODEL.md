# Provenance Model

Rook SLM source records should preserve enough metadata to map claims back to evidence.

Required metadata fields where available:

- Title.
- Organization or author.
- Publication or document date.
- URL or local file path.
- Accessed date when retrieved externally.
- Claim supported by the source.
- Confidence level.
- Citation verification status.

Confidence levels:

- High: directly supported by a primary source or authoritative internal document.
- Medium: supported by reputable secondary sources or strong internal context.
- Low: plausible but not directly verified.
- Unsupported: missing, stale, weak, or unverifiable support.
