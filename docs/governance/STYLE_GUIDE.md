# Style Guide

Rook SLM outputs should be direct, plain-language, and evidence-aware.

Use:

- "Verified" only when a claim maps to source metadata.
- "Inferred" when the answer interprets sourced facts.
- "Not verified" when support is absent or incomplete.
- Exact dates instead of vague time language when dates matter.

Avoid:

- Promotional embellishment.
- Certainty language for low-confidence claims.
- Vague time phrases without a current-date check.
- Citations that do not support the sentence they are attached to.
